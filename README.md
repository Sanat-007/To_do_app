# To_do_app

A simple Python-based To-Do web application.  
This project provides a basic task management interface where users can create, view, update, and delete to-do items. It is implemented using a Python web framework and follows a modular project structure.

## Features

- Create new tasks
- List existing tasks
- Update task status
- Delete tasks
- Clean, minimal UI

## Tech Stack

- **Backend:** Python (likely Flask/Werkzeug)  
- **Frontend:** HTML, CSS  
- **Run Server:** WSGI entrypoint (e.g., `wsgi.py`)  
- **Dependencies:** Listed in `requirements.txt`
